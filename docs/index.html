Questions? Answers.

Will it break my Mac?

No. Every action is reversible via Recovery Vault, which keeps everything for 14 days. RepairIQ never touches system files or uses elevated permissions.

Is it really free?

Yes. No hidden costs, no data harvesting, no "free trial" that expires. The full app is free.

Does it work offline?

100%. RepairIQ never connects to the internet. Zero telemetry, zero analytics, zero network requests.

How is it different from CleanMyMac?

We show our evidence. Every recommendation includes a confidence score and a list of reasons why it's safe. If we're wrong, Recovery Vault has your back.

What about Windows?

Full Windows support is included. Same features, same safety guarantees, same 4 MB binary.
